可打开 小储服务端 后注册账号，便于以后管理授权！

# 部署源码

阅读提示

如果您已经做好上面的准备，可以开始阅读了，这里会告诉你如何将小储系统源码部署到服务器！

# 1、准备一台服务器或主机，用于存放网站程序文件

# 您有两种可选择方案

# 方案1（使用自有服务器/主机空间安装）：

1、下载程序源码：
可在 小储服务端 的首页 点击 程序下载 按钮 下载所需的程序版本，下载分为安装包和完整包，安装包的作用是对上一版本的客户端进行升级，如你安装的程序版本是V1.81，那么就去下载 V1.82版本的更新包覆盖升级！
2、将程序源码解压到你的服务器或主机空间内，下方是网页端后台的首页界面，可以点击程序下载按钮下载最新稳定版本程序
3、解析域名到你的服务器或主机，访问安装！

# 方案2（使用小储系统官方托管空间安装）[推荐]：

1、在小储服务端 的扩展中心 打开网页端后台或打开托管商城



2、根据配置需求购买官方托管空间，托管空间均可完美运行小储系统，开通托管后记得激活，如果激活不了请及时联系官方群内的管理,如图所示：



3、购买并且激活完成后，可在小储服务端 扩展中心 打开网页端后台进入找到下图的托管服务器列表，或点击我的托管按钮进入：





4、开始部署小储安装包：

Ps：部分国外节点可能部署源码较慢，可以点击刷新按钮直至压缩包文件大小不再变动，任何再点击解压按钮，解压成功后，文件部署完毕！
